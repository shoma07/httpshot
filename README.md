# Httpshot


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'httpshot', git: 'https://github.com/shoma07/httpshot.git',
                branch: 'master'
```

And then execute:

    $ bundle

## Usage

```
$ httpshot <url>
```
